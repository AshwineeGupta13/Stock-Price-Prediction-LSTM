# Stock-Price-Prediction-LSTM

This project is a Stock Price Prediction Web Application built using Streamlit, TensorFlow, and LSTM (Long Short-Term Memory) Neural Networks.

The application allows users to:

* Register and Login securely
* View historical stock data
* Visualize stock trends
* Analyze Moving Averages (100 MA and 200 MA)
* Predict future stock prices using a trained LSTM model
* Forecast prices
